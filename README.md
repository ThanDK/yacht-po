# FLIPS Yacht Investment

Premium fractional yacht ownership website for FLIPS Investment - Phuket, Thailand.

## Yachts

- **Merry Fisher 1095 FLY** - 34-foot flybridge motor yacht with 3 cabins
- **Cap Camarat 10.5 CC** - 33-foot center-console sports yacht

## Tech Stack

- Vite (dev server & build)
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Interactive photo galleries with tab filtering
- Specifications comparison table
- Investment program details
- Responsive design
- FLIPS branding with ocean color scheme
